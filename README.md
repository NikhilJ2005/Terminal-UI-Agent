# Agentic Terminal UI (Backend + Frontend)

An intelligent, real-time AI coding agent platform that marries a terminal-inspired frontend with an autonomous, tool-equipped FastAPI backend. Powered by Google Gemini and ChromaDB, the agent can index repositories, search codebases semantically, read local files, and safely execute terminal commands via a strict ReAct loop.

The architecture leans heavily into modular, single-responsibility files (no file exceeds 80 lines) totaling ~550 lines of lean Python code.
